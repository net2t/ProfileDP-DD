# OutLawZ Profile

A personal portfolio website featuring Urdu/Hindi branding, poetry, projects, and contact functionality.

## Features

- Custom Urdu fonts (Outlawz brand)
- Hero section with brand name
- Projects/Repositories showcase
- Meta information display
- Poetry section
- Bio section
- Contact options (Email, WhatsApp)
- Message modal with Google Sheets integration

## Usage

Simply open `outlawz.html` in a web browser.

## GitHub Pages

This site is deployed via GitHub Pages. To update:
1. Push changes to the `main` branch
2. Go to Settings > Pages
3. Select main branch as source
